# SkillCircle

A modern skill exchange platform built with HTML, CSS, and JavaScript. Connect and trade skills directly—no money, no fees.

## Features

- Landing page with hero and animations
- Browse and post skills
- Real-time search and filtering
- How It Works guide
- FAQ section
- Local data persistence with localStorage

## Tech Stack

- HTML5, CSS3, JavaScript
- Font Awesome icons
- Google Fonts (Fraunces, DM Sans)
- No backend required

## Get Started

1. Open `index.html` in your browser
2. Browse skills or post your own
3. Connect with others to exchange skills

## How to Use

1. Open `index.html` in a modern web browser
2. Click "Get Started" to browse available skills
3. Post your own skills or browse others
4. Click "Connect" to message someone
5. Exchange skills directly with community members

## Browser Compatibility

- Chrome/Chromium (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers
